**Module-level lazy LM init (add before run_analysis):**
```python
_lm_initialized: bool = False

def _ensure_lm_configured() -> None:
    global _lm_initialized
    if _lm_initialized:
        return
    from config import get_settings
    s = get_settings()
    configure_lm(
        openai_api_key=s.llm_api_key,
        model=s.llm_model,
        base_url=s.openai_base_url,
        anthropic_api_key=s.anthropic_api_key,
        consolidator_model=s.llm_model_consolidator,
    )
    _lm_initialized = True
```

**run_analysis() pipeline steps (from 09-RESEARCH.md, follow numbering):**

Step 1 (5%): Mark job status="processing", progress=5
Step 2 (10%): Resolve clinic name — `db.table("sf_clinics").select("id, name").eq("id", clinic_id).single().execute()` → `clinic_name = result.data["name"]`
Step 3 (15%): `_ensure_lm_configured()` (call before any DSPy step)
Step 4 (20%): `ingest_from_evolution(clinic_id, clinic_name)` → `conversations`
Step 5 (25%): Guard — if not conversations: fail job with "Nenhuma conversa encontrada para essa clinica nos ultimos 90 dias." and return
Step 6 (30%): `compute_metrics(conv)` for each conversation → `metrics_list`
Step 7 (35-70%): `analyze_conversation(conv.messages, clinic_name)` per conversation, wrapped in try/except (DSPy failure → append `SemanticAnalysis(error=str(e))`, log warning, continue) → `analyses`
Step 8 (75%): `detect_outcome(conv.messages, clinic_name)` per conversation → `outcome_results` (wrap each in try/except too)
Step 9 (80%): `extract_shadow_dna(conversations, clinic_name, analyses)` → `shadow_dna`
Step 10 (85%): `aggregate_outcomes(outcome_results)` → `outcome_summary`
Step 11 (87%): `compute_financial_kpis(outcome_summary, shadow_dna=shadow_dna, clinic_name=clinic_name)` → `financial_kpis`
Step 12 (90%): `aggregate_metrics(metrics_list)` → `agg_metrics`
Step 13 (92%): `build_blueprint(client_slug=clinic_id, client_name=clinic_name, shadow_dna=shadow_dna, outcome_summary=outcome_summary, financial_kpis=financial_kpis, agg_metrics=agg_metrics, analyses=analyses, generated_at=datetime.utcnow())` → `blueprint_dict`
Step 14 (95%): INSERT to la_blueprints: `db.table("la_blueprints").insert({"job_id": job_id, "clinic_id": clinic_id, "blueprint": blueprint_dict}).execute()`
Step 15 (97%): infer resources — wrapped in try/except, MUST NOT re-raise:
```python
try:
    infer_and_persist_resources(conversations, clinic_name, clinic_id, job_id, shadow_dna, db)

```
Step 16 (100%): Mark job done: `_update_job(db, job_id, status="done", progress=100, current_step="Concluido")`
