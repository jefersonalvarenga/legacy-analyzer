Evolution Message row structure (from RESEARCH.md, verified against Prisma schema):
```python
# row["key"] JSONB example:
key = {
    "remoteJid": "5511912345678@s.whatsapp.net",  # individual
    "fromMe": False,  # True = clinic sent, False = patient sent
    "id": "3EB073ADB8DD86D47345ABD"
}
# row["message"] JSONB example:
message = {"conversation": "Olá, quero agendar!"}
# or:
message = {"extendedTextMessage": {"text": "Clique aqui: https://..."}}
# or (media):
message = {"audioMessage": {"url": "...", "seconds": 12}}

# Other row fields:
# row["pushName"]         — str | None — patient's WhatsApp display name (for fromMe=False only)
# row["messageType"]      — str — "conversation", "extendedTextMessage", "imageMessage", etc.
# row["messageTimestamp"] — int — Unix epoch seconds
# row["instanceId"]       — str — FK to Instance.id (UUID/CUID)
```
